# project-SnakeCharmers01

# 🤖 Персональний асистент

CLI-додаток для керування контактами і нотатками з інтелектуальним автодоповненням команд та сучасним інтерфейсом.

---

## 🚀 Встановлення та запуск

Ви можете використовувати цей застосунок двома способами:

### Спосіб 1: Встановлення як пакет (рекомендовано)
Це дозволить запускати помічника командою з будь-якої папки вашої системи:

1. Склонуйте репозиторій та перейдіть в папку проєкту.
2. Встановіть пакет:
   pip install .
3. Запустіть програму командою:
   helper-bot

### Спосіб 2: Запуск як скрипт
1. Встановіть необхідні бібліотеки:
   pip install -r requirements.txt
2. Запустіть головний файл:
   python cli.py

---

## 📁 Структура проєкту

* **cli.py** — головний модуль, що керує навігацією.
* **contacts.py** — реалізація адресної книги та валідація.
* **notebook.py** — реалізація нотатника та робота з тегами.
* **utils.py** — допоміжні інструменти (Tab-підказки).
* **setup.py** — конфігурація для інсталяції проєкту.

> **Збереження даних:** Програма автоматично зберігає ваші записи у файли `contacts.pkl` та `notebook.pkl` при завершенні роботи.

## 📒 Адресна книга

| Команда                | Опис                                 |
| ---------------------- | ------------------------------------ |
| `add`                  | Створити новий контакт               |
| `edit [ім'я]`          | Редагувати існуючий контакт          |
| `delete [ім'я]`        | Видалити контакт                     |
| `all`                  | Показати всі контакти в таблиці      |
| `search [запит]`       | Пошук за іменем, телефоном або email |
| `birthdays`            | Список іменинників на найближчі 30 днів|
| `show-birthday [ім'я]` | Дізнатись кількість днів до свята    |
| `back`                 | Повернутись до головного меню        |

* **Валідація:** Телефони приводяться до формату `+380XXXXXXXXX`. Email та дати перевіряються.
* **Унікальність:** Система використовує `uuid` для ідентифікації контактів з однаковими іменами.
* **Дні народження:** Обробка 29 лютого та перенесення свят з вихідних на понеділок.

---

## 📓 Нотатник

| Команда              | Опис                                  |
| -------------------- | ------------------------------------- |
| `add`                | Створити нотатку з тегами             |
| `edit [заголовок]`   | Змінити текст або теги нотатки        |
| `delete [заголовок]` | Видалити нотатку                      |
| `all`                | Показати всі нотатки                  |
| `search [запит]`     | Пошук за змістом заголовка або тегами |
| `sort [тег]`         | Групування нотаток за обраним тегом   |
| `back`               | Повернутись до головного меню         |

* **Теги:** Вводяться через кому (наприклад: `python, work, important`).
* **Пошук:** Миттєвий доступ за ключовими словами.

---

## ✨ Особливості та технології

* **Автодоповнення (Tab):** Підказує команди, імена контактів та заголовки нотаток.
* **Rich UI:** Таблиці та кольорові повідомлення у консолі.
* **Надійність:** Обробка помилок вводу без закриття програми.
* **Залежності:** `prompt_toolkit`, `rich`.
