# project-SnakeCharmers01

# Персональний асистент

CLI-додаток для керування контактами і нотатками з автодоповненням команд.

## Запуск

```
pip install -r requirements.txt
python cli.py
```

## Структура проєкту

```
cli.py        — головне меню
contacts.py   — адресна книга (дані + CLI)
notebook.py   — нотатник (дані + CLI)
utils.py      — спільні утиліти: автодоповнення, ask_yes_no
```

Дані зберігаються у `contacts.pkl` і `notebook.pkl` автоматично при виході.

---

## 📒 Адресна книга

| Команда                | Опис                                 |
| ---------------------- | ------------------------------------ |
| `add`                  | Створити контакт                     |
| `edit [ім'я]`          | Редагувати контакт                   |
| `delete [ім'я]`        | Видалити контакт                     |
| `all`                  | Показати всі контакти                |
| `search [запит]`       | Пошук за іменем, телефоном або email |
| `birthdays`            | Дні народження у найближчі 30 днів   |
| `show-birthday [ім'я]` | Скільки днів до дня народження       |
| `back`                 | Повернутись до головного меню        |

**Формат телефону:** `050 123 45 67`, `0501234567`, `+380501234567` — будь-який з них нормалізується до `+380XXXXXXXXX`.

**Номери телефону:** є можливість додавати декілька номерів телефону до одного контакта.

**Однакові імена:** якщо є кілька контактів з однаковим іменем, програма запитає, чи додати ще один контакт з таким іменем, а при видаленні запитає який з них видалити.

**Дні народження:** команда birthdays - показує дні народження на найближчих 30 днів і обробляє вийняток 29.02 в невисокосний рік.

## 📓 Нотатник

| Команда              | Опис                          |
| -------------------- | ----------------------------- |
| `add`                | Створити нотатку              |
| `edit [заголовок]`   | Редагувати нотатку            |
| `delete [заголовок]` | Видалити нотатку              |
| `all`                | Показати всі нотатки          |
| `search [запит]`     | Пошук за заголовком або тегом |
| `sort [тег]`         | Нотатки з тегом — вгору       |
| `back`               | Повернутись до головного меню |

**Теги:** вводяться через кому: `python, робота, важливо`. 

**Пошук:** пошук можливий по тегах та по заголовках нотатки.

---

## Автодоповнення

**Tab:** підказує команди, імена контактів, заголовки і теги нотаток залежно від контексту.
